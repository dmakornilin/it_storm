# компонент работы с Контактами

> ## ContactSection
> ### информация берется из параметров 
> #### phone=BaseParams.phone;
> #### email=BaseParams.email;
> #### address=BaseParams.address;
> ### интерфейсные методы
> **to_email()** - открытие почтового клиента (сервис EmailService)
> **to_consult()** - модальное окно заказа на бесплатную консультацию (сервис ModalNavigateService)
