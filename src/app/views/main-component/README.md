# Компонент для работы с корневой страницей
## Для навигации применены якоря. Последовательно используются компоненты 
> ### ActionsList
> слайдер акций
> ### PriceListSection 
> Предлагаемые услуги (прайс-лист)
> ### AdvantageSection
> блок **Кто мы и чем занимаемся?** Список преимуществ
> ### TopArticlesSection
> Блок **"Популярные статьи"**
> ### ReviewsSlider
> слайдер отзывов
> ### ContactSection
> Блок **"Контакты"**
