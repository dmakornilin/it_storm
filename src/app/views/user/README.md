# Компонеты работы с пользователем

> ## Login
> страница входа
> реализована валидация ключевых параметров 
> если пользователь уже залогинен, идет redirect на корневую страницу

> ## Signup
> страница регистрации
> реализована валидация ключевых параметров
> ### Основные переходы
> #### to_agreement()
> На страницу пользовательского соглашения
> #### to_persdata()
> На страницу согласия об обработке персональных данных
> 

> ## Agreement
> Страница пользовательского соглашения
> ## PersData
> Страница согласия об обработке персональных данных
