# Компоненты работы с заказами (открывает модальное окно)

> ## OrderCard
> ### компонент заказ
> ### основные интерфейсные параметры
> #### ordTypeSelected 
> Вид заказ **order** / **consultation** 
> #### currentOrderTitle
> Заголовок заказа в соответствие с видом
> #### refresh_form()
> обновление формы (для заказа формируется список выбора услуги)
> 
> #### реализована валидация существенных полей
> #### для заказа услуги - отправка на backend с отслеживанием результата
> ####  если пользователь залогинен, по умолчанию проставляется имя 
> 
> ### при успешном оформлении заказа открывается окно "Спасибо" 
 
> ## ThankCard
> модальное окно "Спасибо за вашу заявку"
> вызывается только при успешном оформлении заказа/заявки
>
