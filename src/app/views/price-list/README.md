# Компоненты работы с прайс-листом (список предоставляемых услуг)

> ## PriceListSection
> список предоставляемых услуг
> получен через сервис PriceService
 
> ## PriceListCard
> карточка услуги
> ### входящий параметр card = input.required<PriceListCardType>();
> содержит ссылку на оформление заказа по услуге 
> открытие модального окна (сервис **ModalNavigateService**)
