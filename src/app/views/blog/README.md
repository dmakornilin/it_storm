# Компоненты работы с блоком "Блог"

> ## BlogArticle
> статьи блога
> ## взаимодействие через сигналы
> 
> ### isCommentsLoading<boolean> 
> есть загруженные комментарии
>
> ### isEditable<boolean>
> есть возможность редактировать (пользователь залогинен) 
> 
> ### comments_list
> список загруженных комментариев
> 
> ### totalComments
> количество загруженных комментариев
> 
> ### getArticleInfo
> информация о статье
> 
> ### getRelatedArticles
> список связанных статей
>
> ## основные методы
> 
> ### refresh(u: string)
> первичная загрузка статья и основная иформация (связанные статьи)
> 
> ### load_next_comments()
> загрузка следующей порции комментариев (до последнегл)
> 
> ### post_comment()
> запись комментария
