# ----  Общие сервисные директивы

> ### ClickOutsideDirective
> 
> директива обработки click вне компонента
> 
> используется при прорисовке выпадающего меню 
> (закрытие при клике вне меню)
> 
 
> ### PasswordRepeat
> 
> валидация повторногго ввода пароля
> (оставлена на перспетиву в проекте не используется
> в форме нет поля повторного ввода пароля при регистрации)
