# ----  Общие сервисные компоненты

## MainMenu 
> ### Компонент меню - используется в header и footer
> 
> основной функционал - отрисовка стандартного меню
> при клике - выделение активного пункта
> и вызов сервиса навигации
> 
> для управления использубтся индексы пунктов меню
>
> ### export const MENU_NOTHING_INDEX
> не активен ни один пункт
> 
> ### export const MENU_SERVICE_INDEX
> 
> пункт "Услуги"
> 
> ### export const MENU_ABOUT_INDEX
> 
> пункт "О нас"
> 
> ### export const MENU_ARTICLE_INDEX
> 
> пункт "Статьи"
> 
> ### export const MENU_REVIEW_INDEX
> 
> пункт "Отзывы"
> 
> export const MENU_CONTACTS_INDEX
> 
> пункт "Контакты"
