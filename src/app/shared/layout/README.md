# ----  Блок объединяет header и footer

> ### Header
> 
> использует компонент **MainMenu** вызывает с флагом **[isNavigated]="true"**
> 
> Примечание: Поскольку базовый цвет **container-phone**  
> **var.$item-active-color** при наведение на активные элементы 
> используется выделение цветом **blue**
> 
> использует навигацию модального окна **ModalNavigateService**
> ### toPhoneConsult()  
> заказать бесплатную консультацию по телефону
> 
> ### При клике на элемент приветствие
> 
> если пользователь не залогинен - переход на форму логина
> 
> если пользователь залогинен - выпадающее меню с пуктом **logout** 
>


> ### Footer
>
> использует компонент **MainMenu** вызывает с флагом **[isNavigated]="false"**
>
> использует навигацию модального окна **ModalNavigateService**
> ### to_consult()
> заказать бесплатную консультацию по телефону
>
> 
> ###  to_email()
> при клике на элемент **email**
> открывает почтовый клиент 
> c темой **"Заказать бесплатную консультацию"** 
>
